¡Bienvenido al **Repositorio de Prompts Educativos**! Esta es una aplicación web en formato blog diseñada específicamente para la comunidad docente y educativa. Su objetivo principal es centralizar, organizar y facilitar el acceso a instrucciones estructuradas (*prompts*) optimizadas para Modelos de Lenguaje (como ChatGPT, Claude, Gemini, etc.), ayudando a los profesores a integrar la Inteligencia Artificial de forma práctica y efectiva en sus aulas.

🚀 **Ver la aplicación en vivo:** [https://juanmale14.github.io/repositorio-de-prompts-educativos-Blog-./](https://juanmale14.github.io/repositorio-de-prompts-educativos-Blog-./)

---

## ✨ Características Principales

- **Catálogo Organizado:** Filtra y busca prompts según el objetivo pedagógico (creación de rúbricas, situaciones de aprendizaje, adaptación curricular, diseño de actividades, dinamización de tutorías, etc.).
- **Diseño "Copy & Paste" Rápido:** Interfaz limpia con botones o bloques optimizados para copiar las instrucciones al portapapeles con un solo clic.
- **Enfoque Competencial y Criterial:** Prompts adaptados a marcos educativos modernos (como la evaluación formativa y el lenguaje inclusivo).
- **Formato Web (Blog):** Una navegación intuitiva y ligera, ideal para ser utilizada en tiempo real durante la planificación de clases.
- **Acceso Libre:** Hospedado directamente en GitHub Pages de forma estática y gratuita.

---

## 🛠️ Tecnologías Utilizadas

La aplicación está construida utilizando tecnologías web estándar para garantizar velocidad y portabilidad:

- **HTML5:** Estructura semántica del blog.
- **CSS3:** Diseño responsivo, moderno y adaptado a dispositivos móviles y ordenadores.
- **JavaScript (Vanilla):** Funcionalidades interactivas (motores de búsqueda, filtros por categorías y función de copiar al portapapeles).
- **GitHub Pages:** Plataforma de despliegue y alojamiento.
